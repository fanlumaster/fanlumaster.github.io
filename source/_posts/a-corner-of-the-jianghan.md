说是要截取时间的泡泡，结果因为一些事情耽搁了。近来生活安淡，还是记一记罢。

## Dec 06

早上八点之前醒的。然后也不困了。前一天下午和晚上都打了不短时间的桌球(乒乓)，夜里又和贱荆做那事情，也睡得晚，写博客的缘故。醒了之后，贱荆一直往我身上贴，搞不清是怎么回事儿，但我却是有心无力了，只好中途作罢，晚上才知道，原来早上冲浪翻过了我的博客，有时她会说，用这种方式确认我几点上床。

起来给我煎了糍粑，我要的原味，又给我冲了一杯麦片。中午被我薅去一起去菜市场，因为我要求刺儿少，就买了鲈鱼，12 块钱一斤，一条不大的鲈鱼，16 块钱。又买了两斤多排骨，18 块钱一斤，又买了一点山药、西红柿、青椒、红椒、草鸡蛋、生姜、平菇、小葱。中午烧的鲈鱼，贱荆觉得淡，这是好事儿，她觉得淡，那必然是对我刚刚好。又把前一天忘了用来下面条的菠菜炒了，两个菜，够了。每天吃饭的必备节目是电视剧最好的我们，一开始看觉得傻，后来可能是被同化了，感觉好像也还可以，高中的生活拍的还算真实，只是，相对于我们当初暗无天日的监狱般的灰色生活，还是要好不少的。下午固定活动，打桌球，和牛牛老兄打了一会儿，这老哥周末好像要和朋友开黑，就只打了一个小时多一点，刚好贱荆午睡起了，再薅过来让她锻炼锻炼，也给她入门了一点正手弧圈的动作。晚饭吃排骨，就我来做了。刚换的这个新的燃气灶，确实好用，烧起排骨，比以前快多了。煮排骨的时候发现，玉米、胡萝卜忘买了，又去广华的超市采购了一点，回到家洗完切段放锅里头，最后十几分钟，再剥点蒜，切末，完事儿了，排骨也烧好了。不知道为什么，明明焯完水之后是用热水洗的，结果，有些肉咬起来还是有点柴，大概是离骨头太远，就没有那么 Q 弹软了。

昨天今天都是使用的 antigravity 来编写浏览器新标签页的前端代码，真特么好用。这下完全是面向输入法编程了，我负责打字，然后 agy 负责给我做需求、写代码，对于这种简单的前端项目，还真有那么一回事儿。我这人，就是容易被事情分心，不过，不处理好周边的杂事儿，又怎能专心去搞主线的任务呢？就像这个新标签页插件，前两天那个 Infinity New Tab 的事情，现在是完全不想使用三方的新标签页插件了，刚好拿 agy 写一下，过几天还可以去 B 站分享一波，前两天吐槽的 Infinity New Tab 的视频被他们私信要求下架了，那我发布一个我自己写的插件的事情，总没有理由来要求我删除视频了吧。刚好我的需求很简单，就是一个搜索框，和几排简洁的 shortcuts，交给 agy 来做，试试 coding agent 的能力，结果让我蛮惊讶的，有点用了之后不太能够回得去的感觉。

再之前，是把 C++ 的多线程的一本 epub 拆了，然后用 agy 打开，一边看，一边让 agy 给我补全完整的示例代码，比起我之前一点点去 chatgpt 的网页端去问，也要省心很多。

然后，就越来越懒，懒到连 git commit message 也 agy 来帮我想，嗯，比我自己花更多的时间想出来的还要更贴切一点。
