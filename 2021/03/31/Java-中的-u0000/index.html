Java 中的 '\u0000'

Java 中的 char 类型初始值是 \u0000，这个表示 Unicode 码中的 0000 号字符，但是这个是非打印字符，所以哦我们通过 Java 打印这个字符一般只能看到一个字符。这个在 Java 中一般表示 null char。
