## 十一月零五

快到十一点才起的床。拙荆外出练车，给我留的早餐，多多买菜上买的包子，还有菜市场买的山药，蒸了一下。本以为是蒸的红薯，结果是山药，一戳，脆脆的，还以为没熟呢。这是拙荆念想我肠胃不好，有意整了山药。只是，吃过了蒸红薯，对于蒸山药，就实在难以下咽了，忍着味觉的不适，吃了两大块。一天下来，肠胃确有改善。

午饭拙荆非要亲自下厨，好吧，她的手艺倒也还蛮好，一碟土豆丝，还有一碟小炒黄牛肉。用高压锅蒸的五常大米，大概是六块钱一斤的那种，大概是水分没有控制好的缘故，米饭比起我在油田的时候吃的十月稻田的要差一些，不过，依然是比外面的饭馆要好吃许多的。前一天晚上的蛋炒饭吃得撑了，剩下两只大闸蟹没吃，中午就顺势吃了一只。第一次用吃蟹的工具，解锁了新体验。以前在家都是吃母亲做的那种辣味儿的大闸蟹，而且，乡野习惯，都是直接啃，以前也年轻，不管是吃虾还是吃蟹，有些时候，有些壳儿都可以嚼碎吃掉的，顺势补钙不是。这回儿整的清蒸的，倒是确实很适合拿那个工具剪一下，掏一下了。

下午惯例本来是要写会儿程序的，结果，需要先理一下思路，就大概画了三张草稿纸不到的样子，一看时间，也不早了。晚上七点半去保利国际打会儿乒乓球，差不多九点半回家。该说不说，上一次打球的肌肉拉伤还没有恢复，很影响反手的发挥，不是很得劲儿。新认识一个球友，打球的路数和我原来拿直拍的时候很像，接不好我拉过去的弧圈，尤其是反手拉过去的前冲，我年轻时候也接不好这种，而且，那会儿我的正手弧圈其实也才刚入门哩。

## 十一月零六

今日午后悟出了一个道理，那就是，男人在二十二岁之后，并非可以想嗯就嗯了。那么，可以推算，黄金时代确是二十一岁无疑。小波诚不我欺。

前一天晚上在一个输入法的 QQ 群里面冒了个泡，结果被拉入了 Rime 的大本营，好多熟人。居戎氏和 fcitx5-macos 的作者自然也都在其中。感觉这个小圈子其实也没有想象中那么冷清。话说回来，作为一个现代的信息工作者，输入法本应该得到更多的关注的。奈何，国人恰是喜欢使用隐私换取便利罢了。可是，也没有那么便利，无非是缺几条鲶鱼呀。

在家宅了一天，大概是十一点起的床，无非就是吃饭、睡觉、写程序、上床，山荆在家做饭，连饭馆都不必出去了。简单的幸福。

晚间母亲通话，又谈家里面前几年做生意的事情，大概是三十来万的投入，还是父亲的老同学，结果被坑了一把。我早先就说，和房地产沾上边，人性不可揣度。又谈兄长工作的事情，这下比起之前在市里面的高中要辛苦多了，毕竟带的学生都是要冲刺清北复交浙这种层次的，节奏相当快，那种压力可想而知。好在，毕竟还是在省内，可以用购物卡给家里买点日用品，给自家买了两桶油，也给外婆家买了两桶，顺带着也买了几箱牛奶，如此一来，没我的事儿倒也不差什么，心安许多。家兄是在苏州的私立任职，又说，公立的学校一年的年薪，在苏州那边，也可以来到十七八万的样子，我说，不愧是大城市。苏州有些家长出手也比较阔绰，偶尔给教师送礼物，就是整个年级全部包揽。再次感叹，不愧是大城市。

又问，缺不缺钱，在长沙租房多少钱，我说多少多少，不缺。再不济，我去做点时间教培，也是可以维持生计的嘛。山荆对这一行可谓轻车熟路，我说，俩人互相搭手，就是自己整一个工作室也未尝不可呀。当然啦，有选择的情况下，我还是更愿意宅在家里面写写程序的，清俗。

还问，衣服是不是没带，天冷了。我说不用担心。心里想，山荆买了这么多衣服，连羽绒服都整上了，也确是没有不方便之处了。

有些时候，还是想要回油田搬设备的。然而，还是坚持一段时间罢。就像长跑一样，连续不间断的一个月的五公里不也坚持下来了么。况且，这段时间就是仅仅使用这一只笔记本编写程序，好像也没有什么效率上的损失，不如说，心绪理顺了，做起事情来一身轻松，心里也轻松，这种状态，做一些程序上的绘图，再合适不过了。

## 十一月零七

又是画 html 的一天。想要把 UI 做得符合我自己的审美，还真不是一件轻松的事情。

十一点左右起床。午饭依然是贱内操持。看到厨房的囧状，商量了一番，以后在长沙这边的话，还是尽量出去吃吧。

下午的时候，油田的楼上打来电话，问我当时买的房子多少钱，我说房子本身是三万六。我问，问这个干嘛，她说要卖房子。这个方式是她的朋友的。她的朋友很早就迁到了四川广安。我问她想卖多少。她说三万来块钱。挂掉电话。我想了一会儿，不如我买下。于是拨回电话。商量了一番，三万三成交。嗯，我没有砍价的经验。晚上，对象和我说，看到了中介的广告，说，挂的价格是三万整。她觉得有点亏，毕竟中介也要收取好几千的费用。我觉得倒是还好，刚好是楼上楼下，对我来说，多个几千块钱的溢价能换取那么多的“内存”来腾放我现在的房子里的家什，还算比较划得来。多了一份六十平的空间，一下子就没有那么捉襟见肘了。只是，目前我没什么钱了。只好向家里面借。对象说，她借我。我说不行，我没赚钱养家就已经冒天下之大不韪了，再去花女人的钱，成何体统。

嗯，只好想想偏门，还是要赚一些钱的。这对生活质量来说很重要。

晚饭就出去吃了。依然是公交新村，我是公交新村的肉丝粉。贱内也想念外面的米饭了，就在旁边的快餐店解决了。今日没有拍照，放一点早些时候的照片吧。

说起来，晚上骑小电驴出去吃饭淋了好大的雨。主要是回来的时候。本来想要去保利国际打球的，雨太大，只好作罢。倒是因此一起逛了旁边的一个平价超市，嗯，真平价。一起逛超市还挺好玩的。以后养了小孩，带小孩一起出来逛超市，就更有感觉了。嗯，得赚钱。

## 十一月零九

不知不觉又过了夜里一点半了。我去。

这次处理的问题倒是比较重要的。tsf dll 注入到 32 位应用的时候，有些历史遗留的程序，会自动缩小窗口，排查了一下，发现是 DpiAwareness 的问题。我之前加了这个感知设置。kimi 还让我继续加，而且，是在 dllAttach 的时候加，有点该死了。最后我把这个设置取消掉，问题就解决了。因为当时的窗口是在 tsf dll 内创建的，没有单独分出一个进程，所以，当时是需要设置这个参数来使用 d2d 绘图的。此外，微软自家的 debugview 32 位(说是 64 位，其实还是 32 位)有点问题，问题主要出在了如果我是使用这个应用来捕获 outputdebugstring 的话，那么，这个进程在打字的时候会卡死，嗯，应该是窗口消息那里没有使用异步带来的后果，这个是老问题了。到时候发布正式产品的时候，把 log 部分做一个开关好了，只有在需要抓虫的时候，再在设置中把这个开关打开。

依然是中午起床。之前还做了个噩梦，梦见他妈的飞机失事，然后，荒地求生，并且被恶势力支配的恐惧，然后迷迷糊糊中感到有人在亲我，刚好梦里要上演春梦的情节，然后就醒了。嗯，果然有人在亲我。我真是，睡个觉也不安稳，汗~ 汗汗汗。

午饭就去百宜坊吃了。吃完回家工作了两个多小时。下午依然是炒菜环节。但是晚上就不行了。问题出在井上面，阎连科说过，古井也有被抽干的一天，嗯，这是对的。晚上就是吃了个炒粉。真特么好吃，贱荆说好像有点淡了，我觉得刚刚好，而且辣椒也放得恰到好处，吃完贱荆做的炒粉，再吃贱荆剩下的南洋大师傅，肚子饱饱的，疑似有点小确幸了。

## 十一月一零

本来十点钟上的床，结果躺床上睡不着，起来吃了点龙眼，喝了杯山药玉米糊糊。主要是不放心那个 caret 坐标的 bug，因为老式程序的 dpi 的问题，搜索了一圈，然后用 kimi 给解决了。在调用线程去切换 dpi 感知，然后再恢复，就可以无痛处理没有 dpi 感知的程序的缩放导致的坐标失真的问题了。

午饭依旧是去的百宜坊。晚饭去的公交新村。中午吃完饭，陪山荆练车去了。导致白天几乎是没有写什么程序。哦，原来是下午回来补觉导致的。昨天夜里没怎么睡好。最近不知怎么滴，老是做一些风马牛不相及的梦。前天是太平洋失事，昨天的，忘了，大概能记得是类似的事情。嗯，有点想念油田的睡眠环境。真想快点回去。不过，白天有人喊起床真是一件幸福的事情。如果是我自己，有些时候可能就继续睡了，这对肠胃不好。

晚饭之前，有一点小事，发生了一点龃龉。现在的阶段已经是完全不会吵架，能互相体谅就多体谅一些。反正不是原则性的问题。我的原则性问题也没有几个，一两个吧。这些都是没有问题了，那么，其他问题又有什么不能理解的呢？生活需要包容。我们的精力都是很珍贵的。为了换取将来的精力和注意力自由。

没有什么其他想要记录的事情了好像。泡个脚，睡觉。

## 十一月一六、十一月一七

在家宅了两天。起来得越来越晚了。心情有点微妙。唉，没有运动是这个样子的，在床上就显得很无力了。过了那个年龄段，就不得不靠运动来维持体力。枣谢。

晚饭吃完，先去的家润多，很荒凉、诡异，在这个闹市区，一家超市居然能如此凋敝，很奇怪。相比于公交新村粉店旁边的平价超市，价格有点过高了。更何况，旁边还有个零食很忙呢。

昨日的肠胃很难受，夜里睡得比较晚。睡前肠胃还是难受得不行。加上天气突然转凉，这边房子这么大，也不好开空调，主要是空调也不是很好，如果是在油田的家里，就可以狠狠开空调了，而且，五楼不是顶楼，天花板没有那么高，所以，空间相对狭促，有多了一层保暖的保障。左右两边应该都有暖气供应，也可以渗进来一些暖意。这边终究是有些促狭了，一旦被子被贱荆裹了去，我这边就显得很狼狈了。凉。

今天多喝了不少热水。吃完晚饭也在廉价超市买了不少干松的零食回来，加上换了一家自选餐厅，有山药可以吃，肠胃好多了。

下午的时候，其实坐久了肠胃也不好受，拉贱荆去打了会儿乒乓球，舒缓了一些。风大，就纯当是锻炼了。

不对，还是得讲一点开发的日志。这两天其实主要是在梳理需求，实现起来是没有难点，难在画 UI，主要是设置界面的 UI，本来是打算一个单 html 文件一把梭的，然后纯 html css js 一把梭，但是，还没写几个部分，代码就膨胀到接近 2000 行了，这还只是硬编码呀。不行，不能这么随意。就改到了 vite + ts，然后其实还是 vanilla 项目，不打算使用框架，就是单纯用 vite 来把原来一体的代码给切分成独立的很多个文件，一个门类的界面代码，不管是 html、css 还是 ts 都要分成单独的一个个的，这样改起来就轻松多了，不用在繁杂的单文件中爬梳了。昨天晚上和今天大概就是做这个事情。然后，发现可以先使用思维导图把所有的 UI 界面用树状图的形式给先捋出来，应该可以提升一点效率。

按：今日输入法崩溃了一次。应该和昨天崩溃的那一次是类似的问题，就是那个什么劳什子 get_segmentation 的问题，应该是已经被 move 掉的东西又被拿出来用了。哎，基础工夫不到家呀。这一次大意了，本来想着，这一次就把问题给定位出来给处理的了，结果手贱，一下子点重启了。我屮。
