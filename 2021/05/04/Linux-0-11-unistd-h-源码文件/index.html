前言

这学期在学操作系统，一些实验内容主要是使用 Linux C，很多头文件不知所然，总觉得盲目去看别人的代码，连哪个函数需要使用哪个头文件都不知道，还写个锤儿的代码！首当其冲的就是这个 unistd.h，一怒之下，找到 Linux 0.11 版本的源码来看，这样至少知道我使用的库中的函数有哪些是 unistd.h 中声明的。

功能描述

标准符号常数和类型头文件。该文件定义了很多各种各样的常数和类型，以及一些函数声明。（如果在程序中定义了符号 __LIBRARY__，则还包括内核系统调用号和内嵌汇编 _syscall0()）（括号中的内容暂时不理解，暂时按下不表）
