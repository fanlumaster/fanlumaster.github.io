IDEA tricks

搜索相关

  • Ctrl+Shift+N：按文件名搜索文件，相当于 VSCode 中的 Ctrl + P。

窗口操作相关

  • Ctrl + F4：关闭当前的 Tab。

快捷键

  • Ctrl + Alt + L：格式化代码。
  • Alt + 1：toggle 左边的侧边栏，相当于 VSCode 中的 Ctrl + B。
  • Alt + F12：toggle 命令行终端。
  • Ctrl + Shift + A：然后输入 plugin，就会召唤出插件窗口。
  • Ctrl + Alt + B：从接口中的方法跳转到其实现类。
