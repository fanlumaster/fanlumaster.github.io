其他非十进制之间的互转

一个简单的方法是，可以利用十进制作中转。

另一个方法是，我们给相应进制的数加上代表它们的前缀，然后使用诸如 hex() 函数即可。

例如，我们将十六进制 0x12 数转成八进制数

补充

这里补充各个常用的进制的前缀和后缀。

  • 二进制：前缀 0b/0B，后缀 b/B
  • 八进制：前缀 0o/0O，后缀 o/O，有的资料说，八进制的前缀也可以使用单独一个数字 0，但是，这在 Python 中是行不通的
  • 十进制：前缀 无，后缀 d/D
  • 十六进制：前缀 0x/0X，后缀 h/H
