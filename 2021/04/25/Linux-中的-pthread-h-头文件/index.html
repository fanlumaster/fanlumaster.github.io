简介

Linux 下的线程库函数是由 POSIX 标准定义的，成为 POSIX thread 或 pthread。在 Linux 上线程函数位于 libthread 共享库中，因此在编译时要加上 -lpthread 选项。

源代码

Linux 内核版本是 5.4，文件位置 /usr/include/pthread.h。
