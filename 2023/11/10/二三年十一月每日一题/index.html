11 765. 情侣牵手

这题如果是理解了并查集这个数据结构的话，那么，其实是比较简单的。

核心的地方是这个式子，

\[ (N_1 - 1) + (N_2 - 1) + \dots + (N_n + 1) = (N_1 + N_2 + \dots + N_n) - n = N - n \]

其中，N 代表的是节点的总数，也就是情侣的对数(总人数除以二)，而 n 则代表的是连通分量的总数。

这里是把两个可以配对的情侣看成是一个节点，然后，如果实际上这两个人没有坐在一起，那么，就是和其他的情侣节点 union 了在了一起。
